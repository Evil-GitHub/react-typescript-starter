# 更新日志

本文档记录了 Evil Pro 项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.2.0] - 2024-10-15

### 新增

- ✨ 完整的环境变量管理系统
- 🐳 Docker 和 Docker Compose 支持
- ☸️ Kubernetes 部署配置
- 🔧 环境变量验证脚本
- 📚 详细的配置文档
- 🚀 多环境构建脚本
- 🔄 代理配置优化
- 📦 统一的环境配置管理

### 改进

- 🎯 优化项目结构和配置文件组织
- 🛡️ 增强类型安全和错误处理
- 📝 完善的文档和使用指南
- 🔧 改进开发体验和工具链
- 🌍 支持多环境无缝切换

### 修复

- 🐛 修复环境变量加载优先级问题
- 🔧 修复代理配置在不同环境下的问题
- 📦 修复构建脚本的环境变量传递

### 技术栈更新

- ⬆️ 升级到 React 18
- ⬆️ 升级到 UmiJS 4
- ⬆️ 升级到 Ant Design 5
- 🔧 优化 Webpack 配置
- 📦 更新依赖包到最新稳定版本
